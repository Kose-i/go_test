# golang

## compile[実行ファイルができる]

go build ${SRC}

## compile and run[実行ファイルなし]

go run ${SRC}

# format

gofmt -w filename.go

- before
```
var numbers = map[string] int{
  "one":1,
  "two":2,
  "three":3,
}
```

- after

```
var numbers = map[string]int{
	"one":   1,
	"two":   2,
	"three": 3,
}
```
